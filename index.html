Project Scope

Research Foundation of ScreenMind

ScreenMind addresses the growing mental health impact of excessive smartphone use through an integrated AI-based detection and early intervention platform.

Research Overview

Building an AI-Powered Digital Mental Health Companion

Smartphones and social media are deeply embedded in daily life, but excessive usage is strongly linked to stress, anxiety, depression, poor sleep, and reduced focus. Existing tools often provide only screen-time tracking without meaningful mental health insights.

ScreenMind addresses this gap through a multi-module system that combines smartphone sensing, AI-based behavioral analysis, explainable results, and preventive support for healthier digital habits.

Problem Focus Smartphone addiction and well-being
Approach Digital phenotyping with AI models
Outcome Early detection and intervention
01

Research Problem

The project investigates how smartphone behavior, sleep-related data, communication signals, and screen usage patterns can be analyzed with AI to detect early mental health risks and support timely preventive action.

02

Research Gap

Current systems mainly focus on usage statistics, self-reporting, or isolated data sources. They lack holistic, real-time, privacy-aware, and explainable solutions for smartphone-addiction-related well-being issues.

03

Research Objectives

  • Predict mental health risks from screen usage logs
  • Analyze social media interaction patterns for distress signals
  • Detect social isolation and loneliness trends
  • Estimate sleep disruption risks using phone-based sensor data
04

Methodology

The methodology combines passive smartphone sensing, local feature extraction, AI/ML-based analysis, risk scoring, explainable results, and actionable interventions through a mobile-first architecture.

05

Technologies Used

React Native Kotlin Python FastAPI Firebase SQLite LSTM BiLSTM RoBERTa TensorFlow

Key Features

Core Modules of ScreenMind

The system is composed of four intelligent modules that work together to analyze behavior, detect risks, and provide actionable insights.

Module 01

Mental Health Predictor

Uses screen usage logs and behavioral patterns to estimate mental health risk levels using trained machine learning models.

  • Usage-based risk scoring
  • Time-series pattern analysis
  • Explainable predictions
Module 02

Social Media Analysis

Analyzes interaction patterns and sentiment signals from social platforms to identify early signs of distress.

  • Transformer-based sentiment
  • Interaction frequency analysis
  • Behavioral signal extraction
Module 03

Isolation & Loneliness Detection

Detects reduced communication, mobility, and engagement patterns to identify possible social isolation risks.

  • Communication decline tracking
  • Mobility signal analysis
  • Behavioral change detection
Module 04

Sleep Disruption Estimation

Estimates sleep quality and disruption risks using smartphone usage at night and device interaction signals.

  • Night-time usage detection
  • Sleep pattern inference
  • Risk scoring model
