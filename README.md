## 📌 Project Overview

**ScreenMind** is an AI-powered research project designed to detect early signs of mental health risks using smartphone behavioral data. The system leverages **digital phenotyping**, **machine learning**, and **behavioral analytics** to analyze user interactions and provide predictive insights.

---

## 🎯 Objectives

* Predict mental health risks based on smartphone usage patterns
* Analyze social media behavior for emotional distress detection
* Detect social isolation and loneliness trends
* Estimate sleep disruption using device interaction data
* Provide early intervention insights using explainable AI

---

## 🧠 Core Modules

### 1. Mental Health Predictor

Uses smartphone usage logs and behavioral patterns to estimate mental health risk levels using machine learning models.

### 2. Social Media Analysis

Applies NLP and transformer-based models to detect emotional distress from user interactions.

### 3. Social Isolation & Loneliness Detection

Identifies reduced communication, mobility, and engagement patterns.

### 4. Sleep Disruption Estimation

Analyzes night-time device usage to infer sleep quality and detect disruptions.
